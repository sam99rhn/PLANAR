Quick Tutorial

1. Start with S: Start Graph to create a triangle

2. Use A: Add Vertex to add vertices by selecting two peripheral vertices

3. Use R: Random Vertex for automatic vertex addition

4. Drag to pan, use mouse wheel to zoom

5. Toggle between index/color display with T: Toggle Display

● Ready to start. Click 'S: Start Graph' to begin.
